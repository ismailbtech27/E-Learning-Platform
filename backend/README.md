# Online Learning Platform - Backend

This is a Spring Boot application using Maven.

## Prerequisites
- Java 17+
- Maven (or run via IDE)

## How to Run

### Option 1: Using an IDE (Recommended)
1.  Open the `backend` folder in IntelliJ IDEA, Eclipse, or VS Code.
2.  Let the IDE import the Maven project (`pom.xml`).
3.  Run the `OnlineLearningPlatformApplication.java` class.

### Option 2: Command Line
If you have Maven installed:
```bash
mvn spring-boot:run
```

The server will start at `http://localhost:8080`.
